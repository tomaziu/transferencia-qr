# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-25

### Added

- Local file transfer from phone to computer over Wi-Fi.
- QR Code generation for quick mobile access.
- Real-time upload progress on the desktop panel.
- Resumable chunked uploads.
- Configurable destination folder on the desktop.
- Hosted demo support with temporary server-side downloads.
